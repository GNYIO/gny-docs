# Blockchain

The gny-experiment (opens new window) is the main GNY repositoy. It contains the code for the GNY Blockchain project. The GNY Blockchain is written in TypeScript (opens new window) and follows a modular architecture by using lerna.js (opens new window).

The GNY Blockchain consists of the following packages:

  • @gny/main
  • @gny/base
  • @gny/cli
  • @gny/client
  • @gny/database-postgres,
  • @gny/ed
  • @gny/extendedJoi
  • @gny/interfaces
  • @gny/logger
  • @gny/p2p
  • @gny/protobuf
  • @gny/transaction-pool
  • @gny/type-validation
  • @gny/utils
  • @gny/web-base
  • @gny/web-ed
