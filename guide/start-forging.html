# Start forging

After you have finished:

  • Set Username
  • Register as Delegate
  • Lock account for Voting
  • Vote for Delegate

TIP

You need to change the configuration for your node and add your delegate secret. See Configure/Start-Forging

WARNING

You need to get into the top 101 delegates in order to be able to forge. Check the /api/delegates endpoint
